# 📖 Overview

Spendly is a full-stack personal finance management platform built to help users manage their financial life through an intuitive and modern interface.

It combines secure Google Authentication, cloud persistence, intelligent analytics, budgeting, financial goals, and real-time dashboards into a seamless experience that works across desktop and mobile devices.

Unlike traditional expense trackers, Spendly focuses on delivering a polished SaaS-like experience with persistent cloud storage, responsive design, and dynamic branding.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Gautkk30/spendly.git
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```
